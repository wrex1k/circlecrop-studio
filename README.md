## ✨ What is CircleCrop Studio?

**CircleCrop Studio** is a lightweight web app for creating clean circular crops from photos and videos.

It supports common image formats, HEIC / HEIF conversion, ZIP batch workflows, video trimming, background selection, and browser-based video export.

---

## 🔥 Key Features

- Circular photo cropping with adjustable size
- Rounded-square and full-circle crop control
- HEIC / HEIF image support
- ZIP upload with batch image gallery
- Video crop overlay with trim handles
- Video preview before export
- MP4 and WEBM video export
- Background color selection for exports
- Original, 512 px, 1024 px, and 2048 px image output sizes
- Dark / Light theme support
- Paste, drag-and-drop, and file picker upload

---

## 🚀 How to Run

1. Download or clone this repository
2. Open `index.html` in your browser
3. Upload a photo, video, or ZIP file
4. Crop, preview, and export

**Recommended browser:** Latest Chrome, Edge, or another modern Chromium browser.

> **Note:** Video export support depends on your browser's `MediaRecorder` capabilities. MP4 is used when supported, and WEBM is available as a browser-friendly format.

---

## 🎬 Video Export

- **MP4** - universal standard for sharing across phones and computers
- **WEBM** - web/browser standard with strong modern browser support

Exports include the selected trim range, crop position, background color, and volume settings.

---

## 🛠 For Developers

**Tech Stack:**

- HTML
- CSS
- Vanilla JavaScript
- JSZip for ZIP processing
- heic2any for HEIC / HEIF conversion
- Canvas API for image and video rendering
- MediaRecorder API for video export
